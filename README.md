# GreenNest - Plant Store

A simple React-based web application for browsing plant, signing up and logging in with Google authenticatoin

## Techonologies Used:
--React
--React Router
--Google/Firebase Authentication
--Tailwind CSS
--Daisy UI
--React Icons
--React Toast
--Vite
--HTML, CSS & JavaScript

## Features:
--User signup wih google authentication
--Google login
--Protected Routes (only viewable when logged in)
--Password validation with feedback
--Profile update with name and photo
--Responsive UI with Tailwind and DaisyUI
--Toast notification for success/error messages
